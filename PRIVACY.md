# Privacy Policy

## Overview

Cobalt Shield respects user privacy and is committed to minimizing data collection.

This project is designed with a **privacy-first mindset**, focusing on transparency, simplicity, and control.

## Data Collection

Cobalt Shield does **not collect personal data by default**.

Data may only be collected when a user:

- Submits a contact form
- Initiates direct communication (e.g., email)

The information collected may include:

- Name (if provided)
- Email address
- Message content

## Purpose of Data

Collected data is used strictly for:

- Responding to inquiries
- Communication related to projects or collaboration

Data is **not sold, shared, or used for advertising**.

## Data Storage

- Data is handled through third-party form/email services (e.g., Formspree or email providers)
- Cobalt Shield does not maintain a dedicated user database
- Data retention is limited to what is necessary for communication

## Cookies & Tracking

Cobalt Shield does **not use tracking cookies or analytics tools** at this stage.

If this changes in the future, this policy will be updated accordingly.

## Third-Party Services

Some services may be used to operate the site:

- Hosting providers (e.g., GitHub Pages)
- Form handling services (e.g., Formspree)

These providers may process data according to their own privacy policies.

## User Rights

Users have the right to:

- Request deletion of their submitted data
- Request clarification on how their data is used

Requests can be made via email.

## Security

Reasonable measures are taken to protect data, but no system is completely secure.

Users should avoid sharing sensitive information through public forms.

## Updates

This policy may be updated as the project evolves.
